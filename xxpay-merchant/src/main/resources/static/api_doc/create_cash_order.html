接口名称：代付

支付类型：CASH

请求方式：POST

请求地址：http://domain:3030/api/cash/create_order

代付接口
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
mchOrderNo string(30) 否 商户订单编号，唯一不允许重复
amount string(20) 否 金额，单位分
mchId string(20) 否 商户号
currency string(20) 否 固定cny
number string(30) 否 银行卡卡号
bankName string(30) 否 银行名称
registeredBankName string(50) 否 开户行名称
accountName string(20) 否 账户姓名
mobile string(20) 否 银行卡预留电话
province string(20) 否 银行卡省名称
city string(20) 否 银行卡市名称
idCard string(30) 是 身份证号
返回参数
接口参数 类型 是否可空 参数说明 其他
retCode string(10) 否 (返回状态码).SUCCESS/FAIL此字段是通信标识，非交易标识，交易是否成功需要结合resCode来判断
resCode string(10) 否 业务结果，SUCCESS/FAIL
retMsg string(10) 否 (返回状态码)返回信息，如非空，为错误原因 签名失败 参数格式校验错误
