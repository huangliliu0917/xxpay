接口名称：代付订单查询接口

请求方式：POST

请求地址：http://domain:3030/api/cash/query_order

查询参数
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
mchOrderNo string(30) 否 商户订单编号
mchId string(20) 否 商户号
返回参数
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
retCode string(10) 否 (返回状态码).SUCCESS/FAIL此字段是通信标识，非交易标识，交易是否成功需要结合resCode和result中state来判断
resCode string(10) 否 业务结果，SUCCESS/FAIL
retMsg string(10) 否 (返回状态码)返回信息，如非空，为错误原因 签名失败 参数格式校验错误
result string(10) 否 订单详情。base64编码的json内容。
result明细
接口参数 类型 是否可空 参数说明 其他
payOrderId string(30) 否 平台代付订单编号
mchId string(20) 否 商户号
mchOrderNo string(30) 否 商户订单编号，唯一不允许重复
amount string(20) 否 金额，单位分
currency string(30) 否 币种
state string(30) 否 支付状态(2-代付成功。)。 0/1代付中。2代付成功。3支付失败。
paySuccTime string(256) 是 支付时间
