接口名称：统一下单接口

请求方式：POST

请求地址：http://domain:3030/api/pay/create_order

请求方法：请使用表单提交的方式，将所需参数放到表单域中，提交到该请求地址即可。

网关接口
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
currency string(10) 否 币种（固定传cny）
mchOrderNo string(30) 否 商户订单编号，唯一不允许重复
subject string(64) 否 商品标题
body string(256) 否 商品描述信息
amount string(20) 否 金额，单位分
clientIp string(32) 是 客户端Ip
frontUrl string(128) 否 用户支付成功后跳转的地址
notifyUrl string(128) 否 商户接收支付结果通知的地址 生产环境该地址需要报备
mchId string(20) 否 商户号
payType string(20) 否 支付方式(参见通道payType。 网关 GATEWAY ,快捷 FAST_PAY ,支付宝H5 ALIPAY_WAP )
param1 string(60) 是 扩展参数1
param2 string(60) 是 扩展参数2
