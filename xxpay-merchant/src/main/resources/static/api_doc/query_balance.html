接口名称：账户余额查询接口

接口说明：商户可查询用于代付交易的资金余额

请求方式：POST

请求地址：http://domain:3030/api/balance/query

查询参数
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
signType string(10) 否 签名类型（固定传MD5）
mchId string(20) 否 商户号
返回参数
接口参数 类型 是否可空 参数说明 其他
sign string(32) 否 请求参数的签名
signType string(10) 否 签名类型（固定传MD5）
mchId string(20) 否 商户号
retCode string(10) 否 返回状态，SUCCESS和FAIL，SUCCESS只在查询成功的时候返回
resCode string(10) 否 返回状态，SUCCESS和FAIL，SUCCESS只在查询成功的时候返回
retMsg string(64) 否 说明信息
time string(20) 否 时间戳
balance string(20) 否 余额
